# SCHEMA — pages

> Jekyll content collections rendered by the dash site.

## Structure

| entry | kind | purpose | rules |
|---|---|---|---|
| `_dash/` | dir | Dash UI pages (portfolio, monitor, analytics boards) | terminal |
| `_pages/` | dir | Static site pages | terminal |
| `_posts/` | dir | Blog posts, `YYYY-MM-DD-slug.md` | terminal |

## Placement

- New dashboard surface → `_dash/`; new standalone page → `_pages/`; dated writing → `_posts/`.
